## 路线一：景德镇+婺源 — 陶瓷与徽派美学（4天）

### 🎯 市场空白
**没有竞争对手提供 4 天英文专属陶瓷路线。** 现有产品要么是 1 日游（$210-256），要么是 6-7 天打包黄山的大巴团（$710-1,545）。

### 💰 建议定价：$1,100-1,400/人（2-4人团）

| 项目 | 费用/人(USD) |
|------|-------------|
| 英文向导 4天 | $320-640 |
| 私车+司机 4天 | $280-400 |
| 住宿 3晚（陶溪川Hyatt + Skywells） | $250-400 |
| 半天陶瓷工作坊 | $80-120 |
| 门票 | $40-60 |
| 餐饮 | $100-150 |
| **合计** | **$1,070-1,770** |

### 🏨 推荐酒店
- **景德镇**：陶溪川凯悦臻选（David Chipperfield 设计，$125-180/晚）— 必选
- **婺源**：Skywells Hotel（300年清代老宅，米其林上榜，$97-179/晚）

### ⚠️ 核心痛点
1. 语言障碍极严重——英文几乎不存在
2. 外国游客不会用微信小程序预约博物馆
3. 分不清真假瓷器，需要行家指引
4. 交通混乱，需要预安排私车

### 🚫 应避开
- 周一（主要博物馆闭馆）
- 国庆/春节（人山人海）
- 陶阳里（性价比差）、瓷宫（摄影师垄断机位）

## 路线二：义乌+永康 — 采购寻源（3天）

### 🎯 市场空白
**没有任何竞争者提供 Yiwu + Yongkang 组合路线。** GL-Open 的 $1,200/5天是最近竞品，但不含永康。

### 💰 建议定价：$1,200-1,500/人（2-3人团）

| 项目 | 费用/人(USD) |
|------|-------------|
| 专业采购代理 3天 | $450-750 |
| 酒店 3晚（Aishang/Borrman） | $90-130 |
| 餐饮 | $100-180 |
| 本地交通 | $90-150 |
| **合计** | **$730-1,210** |

### 🏨 推荐酒店
- **Aishang Best Hotel**（$35-43/晚，9.7分，District 2 对面）
- **Shangri-La Yiwu**（$89-227/晚，高端之选）

### ⚠️ 买家最痛
1. **付款后代理跑路** — 最严重
2. **隐藏加价 20-30%** — 普遍
3. **样品好货不对版** — 批量生产偷工减料
4. **60%摊位是贸易公司，不是工厂**
5. **不懂 FBA 合规要求** — 货到亚马逊仓库被拒

### 🎁 独特优势
- 永康（100km）是"中国五金之都"，全球1/3保温杯产于此
- 永康工厂直采可省 15-30% 中间加价
- 无人做此组合，差异化极大

## 路线三：山西古建 —《黑神话》朝圣（6天）

### 🎯 市场已验证
游戏发布后山西旅游搜索量 **+3,178%**。小西天从年 3 万人暴增到 125 万人。需求真实且持续。

### 💰 建议定价：$1,200-1,500/人（2人团）| $900-1,100/人（4人团）

| 项目 | 费用/人(USD) |
|------|-------------|
| 英文建筑向导 6天 | $480-900 |
| 私车+司机 6天 | $360-720 |
| 住宿 5晚（庭院客栈） | $300-500 |
| 8-10个景点门票 | $120-160 |
| 餐饮 | $90-150 |
| **合计** | **$1,350-2,430** |

### 🏨 推荐酒店
- **平遥**：Jing's Residence（$200-230/晚，清代丝绸商宅，9.8分）— 必选
- **大同**：庭院客栈（$60-120/晚）

### ⚠️ 硬伤
1. **英文建筑专业向导极度稀缺** — 需提前 4-6 周预订
2. **冬季封山/关门** — 最佳季节 4-10 月
3. **临汾住宿条件差** — 需降低预期或调整路线
4. **景点极度分散** — 必须私车，公交车不存在

## 🔑 共同建议

1. **英文向导是核心资产** — 三国语言（中英+法/德）是巨大溢价点
2. **酒店即体验** — 不卖"住宿"，卖"住在 300 年老宅/建筑师设计的陶瓷美学空间"
3. **解决痛点 = 定价权** — 每条路线的差异化都是"别人搞不定的事我帮你搞定"
4. **永康组合是独家** — 目前无人做，应优先强化
